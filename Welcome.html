Welcome to the globus-clients for Mac OS X Installer

This package contains a collection of client tools from the Globus Toolkit 5.2.2. The Globus Toolkit is an open source software toolkit used for building Grid systems and applications. It is being developed by the Globus Alliance and many others all over the world. A growing number of projects and companies are using the Globus Toolkit to unlock the potential of grids for their cause. Installing this package to get started with using the grid right away.

This package is provided by the Initiative for Globus in Europe. (The IGE project (RI-261560) is funded by the European Commission under Framework Programme 7 (2007-2013). http://www.ige-project.eu/

This installer guides you through the steps necessary to install globus-clients 1.0 for Mac OS X. To get started, click Continue.
